REGRAS

Ao usar o botão "ROLL DICE", um número aleatório entre 1~6 é gerado no marcador central. Esse número será adicionado ao "Current" localizado no retângulo cinza abaixo do marcador central do PLAYER. Uma borda PRETA é utilizada para selecionar o PLAYER atual. A cada click no "ROLL DICE", um novo valor é incrementado ao "Current" score do PLAYER atual. Caso o número gerado seja 1, o valor do "Current" será redefinido para 0 e o PLAYER atual perderá a vez. Caso o PLAYER queira computar o "Current" score ao "SCORE GLOBAL", localizado abaixo da TAG "PLAYER 1 ~ 2", ele deverá usar o botão "HOLD". Dessa forma, ele assegura seu score, evitando uma redefinição num eventual "ROLL DICE" de valor 1. Vence o PLAYER que assegurar um "SCORE GLOBAL" igual ou maior que 100!
